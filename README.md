Senior Fullstack Developer
--------------------------

How can a game on Commodore 16 lead to code and developing scalable solutions with The New York Times, Huffington Post or Via? It all started in the summer of '86 when my dad brought home a Commodore 16 with only one game - Space Fortress. I was 8 at the time and after a couple of weeks me and my brother started poking in the memory to increase the number of lives ... and we did it :). This paved a way of my career in the programming - starting with a small game, now I've been working with clients like New York Times and Huffington Post.
